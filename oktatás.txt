verziókezelő rendszer:
cvs, svn, vs verziókezelője, git
dir- adott mappa

cd- change directory könyvtárváltás
cd <könyvtár neve>
cd .. - felfelé lépkedünk a köyvtárban

git init - inicializálja a verziókövetést az adott könyvtárra
git status - mi a fájlok státusza az adott könyvtárban

git add <fájlnév> -- jelöld meg verziózásra az adott fájlt (staging terület)
git rm --cached <fájlnév> -- szedd le a fájlról a jelölést (munkaterületre kerül vissza)
git add -A  --  Az összes figyelt fájlt megjelöli

git log - mutasd a verziókat egymás után

git commit -m "Kezdeti verzió."

git config --global user.name <felhasználónevem> -- beállítja a felhasználónevet
git config --global user.name   --  lekérdezem a felhasználónevem

git config --global user.email <email címem> -- beállítja az email címemet
git config --global user.email   --  lekérdezem a felhasználónevem
